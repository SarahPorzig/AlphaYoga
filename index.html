My Portfolio Site

HTML5 CSS JavaScript jQuery Bootstrap

My Portfolio - the site you are on right now! I created this site as a place to showcase my projects. I have achieved a sleek and sharp look with dark images spanning the width of the page contrasted against a white background. I have made use of jQuery to create seamless transitions across the site. Caramar Estate

HTML5 CSS JavaScript jQuery Bootstrap

This is a site I created as a home page for 'Caramar' Estate. 'Caramar' is located in South West Victoria near the Otways and is surrounded by bush, which is why I chose to include dark greens in the site's color palette. A serif font and shades of gray were utilised to give a soft, peaceful feel reminicent of the countryside. The site includes a fair amount of open whitespace to help the user experience the site with ease. I have included jQuery code to allow the user to traverse the site smoothly and a Bootstrap carousel in the center of the page to showcase photography from the property. The site is pleasing to use across all platforms thanks to Bootstrap and CSS @media queries.
